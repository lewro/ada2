A menu displays a list of choices. They are hidden by default and get displayed when the user either clicks/taps or hover over a predefined element. They are often used in primary application navigation. Menu elements can consist of text or text with an icon. A simple menu is only one level deep while complex menu can contain cascading menus where parent element in the main menu contains a secondary menu which appears when a user hovers over that element.
